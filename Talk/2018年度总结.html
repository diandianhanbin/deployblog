                2018年度总结

                sven_weng 2019-01-31 23:52:20 个人总结

                # 总评

                2018年对我来说，是一个有惊喜，有遗憾的一年。

                从去年定的目标来看，很遗憾，基本上都没完成。总体的阅读量下降了太多。写作方面虽然与去年保持持平的状态，但是缺乏深度思考的作品。

                但是换一个角度看，又是一个有惊喜的一年，今年完成了一件人生大事，一个人生大目标，从这方面看，又是非常惊喜的一年。

                # 2018预期目标

                2017年给自己定的目标看起来好像也不太多。

                • 测试管理平台接入性能测试工具
                • 完成任务统一调度系统
                • 多去几个地方走走
                • 读五本实体书
                • 读十本电子书
                • 重写博客系统
                • 机器学习达到使用水准

                但是也只是大概完成了一半。比较实打实的完成的，就是重写博客系统、任务统一调度系统和读五本实体书。最扯淡的目标应该是机器学习吧，不忍直视- -！。

                # 读书与写作

                今年全年都在一个忙碌的状态。阅读量急剧下降，以下：

                • 蒙古帝国史
                • 蒋介石传
                • 易中天中华史-风流南宋
                • 中国通史
                • 袭击面包店
                • 康熙大帝
                • 马关议和之伊李问答
                • 未来简史
                • 丝绸之路
                • 哈利波特与魔法石（原版）
                • 易中天中华史-铁血蒙元
                • 今日简史

                还是以历史为主，去年读了大概30本，今年15本都不到。总体的读书类型也过于局限在历史，相比于去年也有缩水。值得一提的是，读五本实体书这个目标本来是遥遥无期的，亏了我厂的图书馆，让我在今年的最后三个月完成了这个目标。但是现在来看，我自己的阅读观以及历史观跟以往相比，有了很大的改变。看书比以前挑了，很多普通的书籍已经看不上眼了，比如今年大热的《今日简史》、《原则》等，我都觉得写的非常一般。历史方面也感觉很明显，以往看的历史书更多是以故事为主，如今慢慢的开始向文化、制度的方面去学习。

                写作方面基本上与去年持平，但是今年基本上是以学习记录为主，并没有过多的深度思考，比如两年前写的《平行解体》这类文章，今年连在脑海里都没有出现过，希望明年自己在这方面能够让自己满意。

                # 工作

                去年说的一句话，今年又说了一次，今年又换了一份工作，同样的持续着忙碌的状态。不过从跳往鹅厂，也算是完成了一个大目标，能进入行业的头部企业，也算是对能力的一个认可吧。

                去年上半年的状态基本上是把学会一门编程语言变为构建一个测试体系，以及针对问题提出一定的解决方案。而现在，我更希望自己能够进行一定的沉淀，把一些存在的泡沫消化掉，真正的把基础打牢。

                进鹅厂之后就感觉的很明显，身边的人都非常的优秀，很多优秀的思维，能够给我带来很多学习的内容。近朱者赤这话说的确实没错，以前工作中很多未关注的点，突然发现也都是测试的范围，只是以前很多时候没有注意。同时很多工具，思路，都给予我很大的启发，以前总是觉得自己做的一套东西能够大幅度的提高效率，过来才发现，很多地方还有更大的提升空间，比如之前一直都没有涉及到的环境治理这块，真的是非常有学习价值。

                在大厂上班，总会被说成是螺丝钉，这话说起来也对也不对。在大厂上班确实是很典型的，每个人都负责自己一个很小快的内容，仅指望工作去学，毕竟少，但是大厂很多现成的东西，都是可以观摩和学习的，现阶段工作中，我觉得思路和方向，比具体如何实施更为重要。一般而言，只要有了方向，落地仅仅是一个时间问题。而大公司的业务庞大，所有小公司最终的方向都是往大公司靠拢，那么大公司做事的一些方式、视野就能够带来受用终身的学习对象。

                工作方面期待19年能够有更多的学习，能够更加深入到系统底层， 比如Linux、数据库和协议等内容，目前都是比较欠缺的。今年为了弥补这块内容，在极客时间上买了一些课程，不过这部分还没有学完，期待19年能够把已买的三门课程学完。

                # 生活

                生活上还是有所变化的。毕竟今年完成了人生大事，用行话来说，就是步入老年了。我自己的感觉也挺明显的，活动的参与热情没有以前高了，运动量也明显下降了，而且熬夜也熬不住了，以前两三点睡觉第二天都能正常上班，现在1点左右就必须要入睡了，否则第二天整个人都是萎靡不振的。

                不过下半年的生活方式确实做了一些改变，比如现在开始，我每天都会去跑个三公里。从开始到现在已经跑了几个月了，目前来看坚持下去并不难，所以19年我给自己定了一个小目标，全年至少要坚持100天，每天跑上3公里，能多的话当然更好。

                生活上还有一个点变得比较令自己不满意，今年看综艺的时间变长了，以前几乎不怎么看综艺，现在看的似乎变多了，每次加班回来，连游戏都不想打了，就想好好躺着看，然后看一群人在电视上做一些很傻的事情，这种放松似乎最能让自己轻松，也许这就是综艺存在的价值？

                今年其实还萌生了一个奇特的想法，我有没有可能出一本书呢？我一直是一个有好奇心的人，这种事情听起来就非常的刺激，但是写书需要有比较深厚的功底，我不希望自己写出来的东西，自己都看不上。短期应该是指望不上了，这个想法也许可以等自己不那么忙之后，去做一个大胆的尝试。

                随着加班量的增加，现在很多时候做事还是偏向熬夜，以至于养成了这个坏习惯，即使没事的时候，也都不会早睡，这个习惯希望在19年能有所改善。

                # 其他

                很多事情都想做，奈何心有余而力不足。比如博客系统依然没有完全的前后端分离，现在我又想在明年用Go重写掉博客的后台系统。再比如刷刷LeetCode，每次想打开刷两题的时候，总会由于各种原因放弃。

                今年的技术书看的也比以前少了。虽然书单上是没有列技术书，但是一直有断断续续的在看，现在不需要像前两年那样非常系统的去学习某些框架，不过在具体应用的细节上经常会寻找一些新的优化点，所以也就没有完整的看完某本书，也就导致了零零碎碎的仿佛看了一些，但是又没有很大的关联性。

                今年年底的时候，突然开始要收集代码片段。想法的起源是看到一个段子说某位程序员在硬盘里存储了很多代码片段，需要写需求的时候就拿这些片段组起来，改一改就完事了。联想到我自己日常的做法不就是这样吗？只不过我没有单独的硬盘去存这些代码片段，每次都是靠百度。因此年底的时候在博客单独啦了一个代码片段的分类，用来归集日常工作中会使用的代码片段，这样尝试着收集个几年，是否可以让自己编码的效率提升？

                # 小目标

                新的一年，依然再立一些小目标吧！

                • 看20本书（不分纸质电子）
                • 坚持100天每天3公里的跑步
                • 刷掉今年花钱买的三个课程（mysql、数据结构、网络协议）
                • 用Go重写博客的后台
                • 写两篇质量比较高的深度思考文
                • 坚持Boxing100天

                已经过了一个月了，希望这些小目标在2020年回顾的时候，是真的能够全部达成。
