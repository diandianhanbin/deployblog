            2019年度总结

            sven_weng 2020-01-07 23:52:20 个人总结

            # 总评

            2019年，可以说是丰收的一年吧，对于我个人来说，可以算是收货满满。当然，也不完全是一切顺利。

            19年，从自身来看，完成了五年计划三项中的两项，算是比较成功。但是从跟去年一样，阅读量也在急速下降。

            # 19年的目标回顾

            • 看20本书（不分纸质电子）
            • 坚持100天每天3公里的跑步
            • 刷掉今年花钱买的三个课程（mysql、数据结构、网络协议）
            • 用Go重写博客的后台
            • 写两篇质量比较高的深度思考文
            • 坚持Boxing100天

            去年给自己的目标其实不多，但是，说实话，依然没有全部完成。比如阅读。

            19年阅读的量确实是比较少，记得的书也屈指可数，不过从另一个角度来看，自己选书开始有一些讲究了，在阅读的时候，也加入了挺多自己的思考，这样来看，也算是比较好的一点。

            19年对于自身的健康的项目，还是完成的比较不错，100天每天3公里这个目标是超额完成了，今年跑步的公里数超过了400。2020年要继续加油。Boxing由于特殊原因，没有达到100天，比较遗憾。

            买的课程中，数据结构和网络协议是刷完了，但是说实话，没有到应用上面试试的话，还是不能确定自己掌握了多少。mysql这块实在是接触的少，暂时放在2020年去攻克吧。

            Golang是个好东西，不过19年忙着没空研究他，仅仅写了一个简单的支持行的web服务，不过稳定性还是挺强的，跑了一年也没发现有啥问题。20年可以看看有没有机会继续研究这块。

            19年平均算下来，一个月都没有写到一篇文章。20年要有所改观。

            # 工作

            工作上还是按部就班的进行着，唯一的区别是能感觉到自己的慢慢成长。 年初为了做Mock中心，认真的学了一部分协议相关的知识，也因此去购买了网络协议的课程，边学边用的情况下，确实是有比较大的提高。

            今年更多的是学习已有的内容，不限于测试的工具建设，开发的代码架构，协议，通讯等等。在Mock中心项目里面进行实际的运用，通讯协议部分参考了已有的组件的uds，分布式架构的设计参考了压测平台,多协议的支持参考了Vue中生命周期的钩子函数，路由规则管理采用了单例模式的设计等等。通过实际的项目，确实是让自己获益匪浅。

            今年还阅读了一些优秀框架的源码，比如requests和unittest。阅读这些源码可以带来很大的收益，今年看我自己去年的代码，感觉特别嫌弃，这应该就是成长的过程吧。

            今年也慢慢的开始弃用一些Python的第三方库。以前特别迷信各种第三方库，有啥问题都是去找库解决。今年开始阅读源码之后，发现有些第三方库写的真的是坑爹。而且每次安装一个库，其实用到的功能非常少，自己的代码功底日渐提高，大部分基础功能自己都能够快速的去实现，也就没必要再多使用第三方库，使用第三方库所带来的问题还是有一些的，用了第三方库，整个迁移成本会很高，而且再数据量大的时候，除了某些知名的第三方库，大部分还不如自己手写一个实现来得快。年底就遇到了这个问题，原来的老代码用的第三方库来解析配置文件，3000条数据花了30s，我自己重新手写后，耗时降低到0.0004s，天知道那个库到底干了啥。

            今年看了一些微服务和分布式架构的文章，结合了自己工作中接触到的分布式架构，似乎对这块有了一些新的领悟。以前觉得那些希望明年能够根据这些已有的理论造一个轮子出来，并且能够真正的推广应用。

            # 生活

            生活上其实挺单调的。有了小孩之后，基本上生活也都围绕着小孩了，平时回来会打打王者荣耀啥的，现在回来也没办法打了，改成了随时可以暂停结束的天天酷跑。游戏时间比以前少了不少。switch的游戏时间也少了好多，感觉它慢慢的要开始吃灰了。

            看书的工具也从手机逐渐的换回了kindle。晚上搞娃的时候总是黑灯瞎火的，看手机总是会感觉到眼睛不舒服，kindle的灯光还是比较柔和的，唯一的缺点就是自从readfree关站后，不能像以前一样白嫖看书了。

            去年觉得自己今年要改善一下晚睡的习惯，现在看来不太现实了，有个娃之后睡眠比以前熬夜还少了。现在只希望娃快快长大，啥时候能睡个整觉。

            出门的时候慢慢对让座有了一些改观，之前见到老人，婴儿，孕妇等会让座，那是教育所致，自己有娃了之后，确实切身体会到了带娃的不方便。尤其是抱着娃好久之后上车，突然有人给我让座，那份谢谢真的是由衷的感激。

            # 目标

            大目标其实挺明确的，五年计划还欠了一个考研，不过看起来这个五年应该是没办法完成了，太多杂七杂八的事干扰了，放到下一个五年来奋斗吧。

            2020年，也同样的定一些小目标：

            1. 看20本书
            2. 坚持200天跑3公里
            3. 好好研究一下数据库方面的知识
            4. 搞定自己设计的基于Python的微服务架构
            5. 提升自己的英语口语水平。
